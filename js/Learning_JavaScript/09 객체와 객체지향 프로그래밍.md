## 9.객체와 객체지향 프로그래밍

JS 객체

- JS 객체와 배열의 비교

	- 공통점 : 컨테이너이다
	- 차이점 : 
		1. 배열은 값을 가지고, 각 값에는 숫자형 인덱스가 있음   
			객체는 프로퍼티를 가지며, 각 프로퍼티에는 문자열이나 심볼 인덱스가 있음
		2. 배열에는 순서가 있음   
			객체에는 순서가 보장되지 않음(보장되는 것 처럼 보여도, 언제든지 바뀔 수 있음)
